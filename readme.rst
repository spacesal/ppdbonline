###################
PPDB Online
###################

Adalah projek untuk menyelesaikan tugas web programming semester 3, berikut projek kami bangun berdasarkan hasil riset pada sekolah Al Biruni. kami menggunakan framework CI 3 dan Bootstrap 5, tentu projek ini masih sangat sederhana tapi kami berusaha untuk membuat projek ini berjalan sesuai apa yang diinginkan dan untuk bahan belajar kami selama ini, terimakasih.

*******************
Server Requirements
*******************

PHP version 5.6 or newer is recommended.

It should work on 5.3.7 as well, but we strongly advise you NOT to run
such old versions of PHP, because of potential security and performance
issues, as well as missing features.
